Скрипт для сортування файлів за каталогами відповідно до їх розширення.

Даний скрипт сортує всі файли, що знаходяться в зазначеній папці, і переносить їх у каталоги за належністю
згідно з розширенням попередньо очищаючи від сміття в назвах файлу, і переписує його латиницею.

Невідомі файли нормалізуються та переносяться до папки MY_OTHER.

Вміст архівних файлів розпаковується в ARCHIVES, створюючи окрему папку з таким же ім'ям.

Для запуску скрипта в терміналі вводимо наступну команду "python3 file_sorter.py trash_folder"
